# DoenaSoft.FolderList.Xml

A .NET library providing XML schemas and data models for handling folder structure representations generated by the ListFolders application.

## Overview

DoenaSoft.FolderList.Xml is a foundational library that defines the XML schemas and corresponding .NET classes for representing hierarchical folder structures. This library is used by the ListFolders ecosystem to serialize and deserialize folder tree data.

## Features

- **Type-safe XML schemas**: Strongly-typed classes generated from XSD schemas
- **Hierarchical data model**: Represents folder trees with nested items
- **Serialization support**: Easy conversion between .NET objects and XML
- **Multi-target support**: Targets both .NET Standard 2.0 and .NET 10.0
- **Lightweight**: Minimal dependencies, focused solely on data models

## Usage

### Basic Example

```csharp
using DoenaSoft.FolderList.Xml;
using System.IO;
using System.Xml.Serialization;

// Deserialize an existing folder list XML file
var serializer = new XmlSerializer(typeof(RootItem));

RootItem rootItem;
using (var stream = File.OpenRead("folder-list.xml"))
{
    rootItem = (RootItem)serializer.Deserialize(stream);
}

// Access the folder structure
Console.WriteLine($"Root folder: {rootItem.Path}");
foreach (var folderItem in rootItem.Items)
{
    Console.WriteLine($"Folder: {folderItem.Path}");
}
```

### Creating and Serializing

```csharp
using DoenaSoft.FolderList.Xml;
using System.IO;
using System.Xml.Serialization;

// Create a new folder structure
var rootItem = new RootItem
{
    Path = @"C:\Videos",
    Items = new[]
    {
        new FolderItem
        {
            Path = "Movies",
            LastWriteTime = DateTime.Now
        },
        new FolderItem
        {
            Path = "TV Shows",
            LastWriteTime = DateTime.Now
        }
    }
};

// Serialize to XML
var serializer = new XmlSerializer(typeof(RootItem));
using (var stream = File.Create("folder-list.xml"))
{
    serializer.Serialize(stream, rootItem);
}
```

## Data Model

### RootItem

The root element representing the entire folder structure.

- `Path` (string): The root directory path
- `Items` (FolderItem[]): Array of child folder items

### FolderItem

Represents an individual folder in the hierarchy.

- `Path` (string): Relative path from the root or parent folder
- `LastWriteTime` (DateTime): Last modification time of the folder
- `Items` (FolderItem[]): Array of child folder items (for nested structures)
